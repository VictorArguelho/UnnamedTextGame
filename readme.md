# Unnamed Text Game

Um jogo de texto (ainda sem nome e sem tema) feito em C# que roda no terminal.

## Requisitos

- .NET SDK 11.0 instalado
- VS Code instalado
- Extensão C# Baixada no VS Code

# Como baixar o projeto

Clone o repositório:

```bash
git clone https://github.com/VictorArguelho/UnnamedTextGame
```

Entre na pasta do projeto:

```bash
cd UnnamedTextGame
```


# Como rodar o jogo

1. Abra a pasta do projeto no **Visual Studio Code**

```
File → Open Folder → UnnamedTextGame
```

2. Abra o arquivo `Program.cs`

3. Use o comando 'dotnet build' no terminal
